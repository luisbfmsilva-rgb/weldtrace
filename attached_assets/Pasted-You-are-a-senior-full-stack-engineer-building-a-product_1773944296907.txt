# ⚙️ CALCULATION ENGINE (DVS 2207-1)

## Pipe Calculation

IF pipe exists:
→ use stored values

ELSE:

wall_thickness = de / sdr

pipe_area = (PI * (de^2 - (de - wall_thickness)^2)) / 4

---

## Unit Conversion

IF machine.unit == "cm2":
→ cylinder_area_mm2 = cylinder_area * 100

ELSE:
→ cylinder_area_mm2 = cylinder_area

---

## Ratio

RA = pipe_area / cylinder_area_mm2

---

## PRESSURES (DVS MODEL)

P1 = RA * 1.5
P2 = RA * 0.2
P6 = P1

---

## TIMES (DVS MODEL)

t2 = 10 * wall_thickness
t3 = 0.01 * de + 3
t4 = 0.03 * de + 3
t5 = wall_thickness + 3

---

## FINAL PRESSURES (INCLUDING DRAG)

pressure_preheat = P1 + drag_pressure
pressure_max_heat = P2 + drag_pressure
pressure_cooling = P6 + drag_pressure

---

# 🧠 WELDING EXECUTION ENGINE (STATE MACHINE)

The welding process MUST be implemented as a strict state machine.

States:

1. PREPARATION
2. PREHEATING
3. HEATING
4. PLATE_REMOVAL
5. PRESSURE_BUILDUP
6. COOLING
7. FINISHED

Transitions must be controlled and validated.

---

# 🔩 STAGE LOGIC

## 1) PREPARATION

Inputs:

* drag_pressure

Rules:

* Facing pressure max = drag_pressure + 25 bar
* If exceeded → FAIL

User must input:

* misalignment
* gap approval (checkbox)

App must display:

* allowed limits

---

## 2) PREHEATING

Display:

* target pressure = P1 + drag
* live timer (count up)
* expected bead height (optional field)

No automatic failure (monitoring only)

---

## 3) HEATING

Auto timer = t2

Rules:

* pressure must NOT exceed (P2 + drag)
* if exceeded → FAIL
* if time < min OR > tolerance → FAIL

---

## 4) PLATE REMOVAL

Auto timer = t3

Rule:

* if exceeded → FAIL

---

## 5) PRESSURE BUILDUP

Trigger:

* when pressure starts rising

System must:

Compute linear ramp:

pressure_target(t) = P2 + (P6 - P2) * (t / t4)

Display in real time:

* target pressure
* actual pressure

---

## 6) COOLING

Timer = t5

Display:

* target pressure
* actual pressure

Rules:

* if pressure drops → alert sound
* if pressure outside tolerance for too long → FAIL

---

## 7) FINALIZATION

IF all stages OK:
→ APPROVED

ELSE:
→ REJECTED + error message
