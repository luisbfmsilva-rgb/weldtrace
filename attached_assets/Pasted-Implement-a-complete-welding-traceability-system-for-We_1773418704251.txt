# 2 — Joint Digital Signature

Create file:

services/welding_trace/weld_trace_signature.dart

Purpose:
Generate a cryptographic fingerprint of the weld.

Import:

dart:convert
package:crypto/crypto.dart

Create class:

WeldTraceSignature

Method:

static String generate({
required String machineId,
required double pipeDiameter,
required String material,
required String sdr,
required List<WeldTracePoint> curve,
required DateTime timestamp,
})

Process:

1 serialize curve

curveJson = jsonEncode(curve)

2 concatenate weld data

payload =
machineId +
pipeDiameter +
material +
sdr +
curveJson +
timestamp

3 compute SHA256

hash = sha256.convert(utf8.encode(payload))

return hash.toString()

The hash will uniquely identify the weld.
